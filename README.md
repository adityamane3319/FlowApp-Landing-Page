# FlowApp Landing Page

This is my responsive landing page project built using HTML and Tailwind CSS.

I created this project to practice:

- Tailwind CSS
- Responsive design
- Flexbox & Grid
- Sticky navbar
- Mobile menu
- Modern UI design

While building this project, I learned how real landing pages are structured and how to make websites responsive for different screen sizes.

---

## Features

- Responsive Navbar
- Mobile Hamburger Menu
- Hero Section
- Feature Cards
- Testimonials Section
- CTA Section
- Responsive Footer
- Hover Effects
- Gradient UI Design

---

## Technologies Used

- HTML5
- Tailwind CSS
- JavaScript

---

## What I Learned

### Tailwind CSS

I learned how to use utility classes like:

```html
flex grid justify-between items-center hover: md: lg:
```

---

### Responsive Design

I learned how to make layouts responsive using:

```html
sm: md: lg:
```

Example:

```html
class="grid grid-cols-1 md:grid-cols-3"
```

---

### Flexbox & Grid

This project helped me understand:

- Flexbox alignment
- Grid layouts
- Spacing
- Responsive positioning

---

### Fixed Navbar

I learned how to make a sticky/fixed navbar using:

```html
fixed top-0 left-0 w-full
```

and how to handle spacing issues after using fixed positioning.

---

### JavaScript DOM

I also learned basic menu toggle functionality:

```javascript
mobileMenu.classList.toggle("hidden");
```
